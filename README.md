# Creating and Working with Ex-post Harmonised Survey Data
  

Welcome to the Creating and Working with Ex-post Harmonised Survey Data Course! This course is designed to introduce learners to ex-post data harmonisation in cross-national survey research, shows you how to apply harmonisation techniques in your own work, and how to use the NextGen Harmonised Data Gateway for your research purposes.

You can start the course via this link: [![LiaScript](https://raw.githubusercontent.com/LiaScript/LiaScript/master/badges/course.svg)](https://liascript.github.io/course/?https://raw.githubusercontent.com/hirschlisa/harmonisation/refs/heads/main/Harmonisation.md)


## How to cite this course
 Nidanur Basturk, Insa Bechert, Aleksa Sandkuhl (2026): Creating and Working with Ex-post Harmonised Survey Data.

See below for information on re-use and licensing.

## Description

We will give you some insights on data harmonisation strategies, explain how to use harmonised cross-national survey data, give you some criteria for the evaluation of the harmonised data for your intended use, and make you familiar with hands-on tools. Our goal is for you to gain theoretical understanding, practical skills, and the ability to evaluate the quality of harmonised measures. 


## How to (Re-)Use the Course

The online modules are written in LiaScript, a Markdown dialect, making it convenient to customize them for your research institution. For assistance, please refer to the LiaScript documentation.
